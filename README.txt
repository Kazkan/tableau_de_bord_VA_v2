1 - Dépendances
Veillez à ce que les librairies suivantes soient installées avant d'exécuter l'application :
 - streamlit
 - st-pages
 - pandas
 - geopandas
 - streamlit_folium
 - folium
 - shapely
 - numpy
 - plotly
 - tkinter
 - shutil
 - subprocess
 - datetime

2 - Exécution en local sur Windows
Dans le terminal de l'invite de commande, entrez la ligne suivante :
python -m streamlit run "C:\Users\Gabriel.NAVENNEC\Symbhi\Site Pôle Ouvrage - mon\répertoire\local\streamlit_app.py"
